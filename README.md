## Description
This command-line application, built using Node.js, Inquirer, and MySQL, allows business owners to manage their company's employee database. It provides options to view and manage departments, roles, and employees, helping business owners organize and plan their businesses.

## Installation
 
To install this project, follow these steps:

1. Clone this ``` git clone https://github.com/marcusperdue/Employee-Tracker-Project.git``` 

2. Install ```inquirer``` please use ```npm i inquirer@8.2.4```

4. (optional) UI improvement installs ```npm install cfonts```  & ```npm install cli-table3``` application may not run without this.

## Usage
 
Use ```node index.js``` to run application
